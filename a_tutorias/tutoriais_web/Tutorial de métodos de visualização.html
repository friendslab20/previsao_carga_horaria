Tutorial de visualização¶

Matplotlib-É uma biblioteca python usada para visualização de dados, a qual foi criada para tentar replicar as técnicas de visualização utilizadas no Matlab.

In [1]:
import numpy as np 
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
%matplotlib inline

Plotando o primeiro gráfico:

primeiro iremos criar uma função qualquer utilizando o numpy para auxiliar.

comandos para criar o gráfico

Um gráfico qualquer-plt.plot(x,y,color=a cor que você quer para o gráfico,label=legenda)

Título - plt.title()

Nomear eixos - plt.xlabe(),plt.ylabel

Legenda-plt.legend()

Plotar o gráfico-plt.show()

In [16]:
x=np.linspace(0,9,15)
y=x**3
y2=x**2
plt.title('primeiro gráfico')
plt.plot(x,y2,label='d')
plt.plot(x,y,label='c')
plt.legend()
plt.xlabel('eixo x')
plt.ylabel('eixo y')
plt.show()

Personalizar o gráfico com mais de uma função: fig,ax=subplots()

ax.plot(x, y, color=cor da reta, lw=largura da reta, ls= tipo de traço, marker=marcador, marksize=tamanho do marcador)

possíveis símbolos de marcador: marcador = '+', 'o', '*', 's', ',', '.', '1', '2', '3', '4', ...

Boxplot¶

Boxplot - é uma ferramenta gráfica para representar a variação de dados observados de uma variável numérica por meio de quartis.

plotando um boxplot:

sns.boxplot(x='dados categórico,y='dados númericos',data=Dataframe)

In [45]:
