tutorial redes neurais¶

Indroduo redes neurais - Uma rede neural um modelo estatstico composto por ns interconectados e separados em camadas que simulam os neurnios do crebro humano. Ao aplicar algoritmos neste modelo, possvel reconhecer padres e detectar correlaes em dados brutos, simulando o aprendizado humano.

Estrutura de um neurnio - O neurnio artificial uma estrutura lgico-matemtica que procura simular a forma, o comportamento e as funes de um neurnio biolgico. Assim sendo, os dendritos foram substitudos por entradas, cujas ligaes com o corpo celular artificial so realizadas atravs de elementos chamados de peso (simulando as sinapses). Os estmulos captados pelas entradas so processados pela funo de soma, e o limiar de disparo do neurnio biolgico foi substitudo pela funo de ativao.

Entradas - podem ser as sadas de outros neurnios, entradas extrenas, um bias ou qualquer combinao destes elementos.

Bias - um erro sistemtico diferentemente de erro aleatrio. Um ou mais componentes do erro sistemtico podem contribuir para a tendncia. Uma grande diferena sistemtica em relao ao valor de referncia aceito reflete-se em um grande valor de tendncia.

Funo de ativao - Ela faz a transformao no linear que fazemos ao longo do sinal de entrada.

Multi layer perceptron-uma rede neural semelhante ao perceptron, mas com mais de uma camada de neurnios em alimentao direta.

keras - O keras uma biblioteca python para deep learning que pode rodar em cima do tensorflow. Ela muito utilizada por ser fcil e rpido para a construo de uma rede neural. Alm de ter diversas outras qualidades, como facilitar a transformao do modelo em um produto.

construindo uma rede neural com keras¶

Sequential¶

Sequential - O modelo sequencial permite inserir camadas de uma rede neural em srie, onde o output da primeira camada serve como input da segunda, e assim por diante.

from tensorflow.keras.layers import Sequential

model=Sequential()

Dense¶

Dense - um dos tipos de camadas do keras, sendo ela a mais utilizada. Ela tem como objetivo calcular uma funo de ativao em conjunto com os dados de entrada e pesos.

from tensorflow.keras,layers import Dense

model.add(Dense(units=,input_shape=(),activation='')

units -nmero de neurnios na camada.

input_shape - tamanho da camada de entrada so as features.

activation - Funo de ativao escolhida para o modelo.

Dropout¶

Dropout - Consiste em uma tcnica para diminuir overfitting do modelo. Que funciona zerando aleatoriamente uma percentagem de neurnios nas camadas da rede neural. No momento de fazer previses, todos os neurnios so mantidos e a rede neural atua como uma grande mistura de sub-redes menores.

from tensorflow.keras.layers import Dropout

model.add(Dropout(0.2))

Compilar¶

loss - a funo escolhida para avaliar o modelo, comparado a perda dos dados de treinos com o de validao . Para modelos de regresso utiliza-se como funo de perda mean_absolute_error e mean_squared_error.

optimizer - Funo que define como os pesos da rede neural so atualizados. Normalmente utiliza-se o sgd e o adam.

from tensorflow.keras.optimezers import Adam

model.compile(loss='mean_squared_error', optimezer='adam')

Fitando o modelo¶

epochs - Iteraes em um conjunto de dados.

batch_size - Tamanho do lote que define o nmero de amostras que sero propagadas pela rede.

validation_split - Seperar uma parte de treinamento para a validao.

history=model.fit(x ,y n,epochs=100,batch_size=10,validation_split=0.2)

Avaliando o modelo¶

Evaluate - uma funo que gerar uma predio para cada entrada e sada e coletar a pontuao, incluindo mdia de perda e qualquer mtrica que tenha sido configurada, como a mean_squared_error.

model.evaluate(x,y)
