tutorial redes neurais¶

Indrodução à redes neurais - Uma rede neural é um modelo estatístico composto por nós interconectados e separados em camadas que simulam os neurônios do cérebro humano. Ao aplicar algoritmos neste modelo, é possível reconhecer padrões e detectar correlações em dados brutos, simulando o aprendizado humano.

Estrutura de um neurônio - O neurônio artificial é uma estrutura lógico-matemática que procura simular a forma, o comportamento e as funções de um neurônio biológico. Assim sendo, os dendritos foram substituídos por entradas, cujas ligações com o corpo celular artificial são realizadas através de elementos chamados de peso (simulando as sinapses). Os estímulos captados pelas entradas são processados pela função de soma, e o limiar de disparo do neurônio biológico foi substituído pela função de ativação.

Entradas - podem ser as saídas de outros neurônios, entradas extrenas, um bias ou qualquer combinação destes elementos.

Bias - é um erro sistemático diferentemente de erro aleatório. Um ou mais componentes do erro sistemático podem contribuir para a tendência. Uma grande diferença sistemática em relação ao valor de referência aceito reflete-se em um grande valor de tendência.

Função de ativação - Ela faz a transformação não linear que fazemos ao longo do sinal de entrada.

Multi layer perceptron-uma rede neural semelhante ao perceptron, mas com mais de uma camada de neurônios em alimentação direta.

keras - O keras é uma biblioteca python para deep learning que pode rodar em cima do tensorflow. Ela é muito utilizada por ser fácil e rápido para a construção de uma rede neural. Além de ter diversas outras qualidades, como facilitar a transformação do modelo em um produto.

construindo uma rede neural com keras¶

Sequential¶

Sequential - O modelo sequencial permite inserir camadas de uma rede neural em série, onde o output da primeira camada serve como input da segunda, e assim por diante.

from tensorflow.keras.layers import Sequential

model=Sequential()

Dense¶

Dense - É um dos tipos de camadas do keras, sendo ela a mais utilizada. Ela tem como objetivo calcular uma função de ativação em conjunto com os dados de entrada e pesos.

from tensorflow.keras,layers import Dense

model.add(Dense(units=,input_shape=(),activation='')

units -número de neurônios na camada.

input_shape - tamanho da camada de entrada são as features.

activation - Função de ativação escolhida para o modelo.

Dropout¶

Dropout - Consiste em uma técnica para diminuir overfitting do modelo. Que funciona zerando aleatoriamente uma percentagem de neurônios nas camadas da rede neural. No momento de fazer previsões, todos os neurônios são mantidos e a rede neural atua como uma grande mistura de sub-redes menores.

from tensorflow.keras.layers import Dropout

model.add(Dropout(0.2))

Compilar¶

loss - É a função escolhida para avaliar o modelo, comparado a perda dos dados de treinos com o de validação . Para modelos de regressão utiliza-se como função de perda mean_absolute_error e mean_squared_error.

optimizer - Função que define como os pesos da rede neural são atualizados. Normalmente utiliza-se o sgd e o adam.

from tensorflow.keras.optimezers import Adam

model.compile(loss='mean_squared_error', optimezer='adam')

Fitando o modelo¶

epochs - Iterações em um conjunto de dados.

batch_size - Tamanho do lote que define o número de amostras que serão propagadas pela rede.

validation_split - Seperar uma parte de treinamento para a validação.

history=model.fit(x ,y n,epochs=100,batch_size=10,validation_split=0.2)

Avaliando o modelo¶

Evaluate - É uma função que gerará uma predição para cada entrada e saída e coletará a pontuação, incluindo média de perda e qualquer métrica que tenha sido configurada, como a mean_squared_error.

model.evaluate(x,y)
